kml2SQL.py

kml2SQL.py is a simple script that transform US Census County KMLs into INSERT Statements for MySQL.

I realize there are proper and robust tools that help with the KML to DBMS ingesting, but since I needed the data for a Database Systems course, which covered both SQL and XLM, I figured it would be a good way to review the material.

Employer Qualifying using Machine Learning

This iPython Notebook outlines how I used simple natural language processing techniques and machine learning in order to determine which companies to visit during the Career Fair just by using the company description.

During the Texas A&M Engineering Career Fair our campus is visited by many organizations looking for the best talent. As much as I would love to spend my entire day talking to all the companies, I only have a small time frame during which I would be able to talk to our visitors. Naturally I would create a short list of which companies I need to ensure I visit with but I wanted to create a tool that would help me narrow my research and maybe even discover new opportunities.
